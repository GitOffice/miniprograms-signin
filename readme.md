## 微信小程序登录态与敏感信息获取功能

### client

* 暂时修改hosts文件：127.0.0.1 abigaleyu.co
* 在使用微信web开发者工具中，在左边菜单选择“项目”--勾选“开发环境不校验域名、TLS版本以及HTTPS证书”

### server

* 进入`server`文件夹
* 启动redis服务：redis-server
* 启动nodejs: npm start
